## SLIDE 2 — The Business Problem *(~50 seconds)*
`[Slow down. This is where you build tension.]`

Let me ask a simple question. /

How fast can we tell a client / what climate change /
will **cost** their building? //

Today, the honest answer is — //

**Slow.** / **Scattered.** / **Disconnected.** //

Here's what it actually takes — / **for every single property.** /

It takes **two full days** / just to assess one building's
climate risk / across floods, heat, wildfire. //

Our teams are stitching together / **eight to twelve**
different data sources / **by hand.** /

And here's the worst part — //

There is **zero** connection /
between the risk we find /
and the strategy we give the client. //

*So what does that cost us?* /

Slower pitches. / Generic advice. /
And clients sitting on risk /
that **no one has put a number on.** //

`[Pause. Let it sink in. Click to next slide.]`

## SLIDE 3 — The Solution *(~55 seconds)*
`[Energy shift. Pick up the pace. This is the "hero" moment.]`

Climate Nexus **fixes this.** //

We score **any building's** climate risk / **in seconds** — /
and deliver the plan to fix it / **in minutes.** //

One platform. / Any address. / A **client-ready action plan.** //

Here's how it works — / three simple steps. /

**Step one: Score the risk.** /
You type in an address — / or upload an entire portfolio. /
In **seconds**, / you get a clear zero-to-hundred score /
across **six** climate hazards — /
flood, / extreme heat, / wildfire, / seismic, /
regulatory risk, / and elevation. /
And each score comes with a **dollar figure** /
showing the financial exposure. //

**Step two: Prescribe the cure.** /
The AI takes that risk profile /
and builds a **ranked action plan** — /
every recommendation is costed, /
with payback timelines /
and matching incentives / like tax credits and grants. //

**Step three: Deliver.** /
You get a **branded, client-ready strategy deck** — /
refined through a simple conversation /
in plain language with the broker. //

This isn't a dashboard that sits on a shelf. /
This is a tool that **goes into the meeting room.** //

`[Click to next slide]`

## 💡 Delivery Tips

> [!TIP]
> - **Slide 2 (Problem):** Slow down. Let each pain point breathe. The audience needs to *feel* the problem before you solve it.
> - **Slide 3 (Solution):** Pick up the energy. You're the hero now. Count the steps with your fingers.
> - **Slide 5 (Value):** Make eye contact when you say "forty-X return" and "zero dollars marginal cost." These are your kill shots.
> - **Slide 6 (Close):** Don't rush the ending. Pause before "Invest in Climate Nexus." Let the room come to you.

> [!IMPORTANT]
> **Practice the transitions.** The click between Slide 2 → Slide 3 is your most important moment. Go from tension to confidence in one beat.
